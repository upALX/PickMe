## How to use ü´Å

**requirements:**
  - Node >= 20 ``sudo apt install nodejs npm``
  - Typescript ``npm install typescript @node/types -D``

**1 - Clone this repo:**
```
git clone git@github.com:upALX/PickMe.git
```

**2 - Install all dependencies:**
``
npm i
``

**3 - Run the app:**
``
npm run dev
``
